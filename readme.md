## TransStatusBarUtil  
4.4.4+ 状态栏透明。  
  
## 使用  
[1] 将工具类集成到你的工程中。  
[2] 在 Activity 的 onCreate() 中：  
<pre>
StatusBarUtil.setStatusBarTrans(this);
</pre>
